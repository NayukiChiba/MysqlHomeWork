# 实验六 存储过程

## 一、实验目的

1. 掌握存储过程的创建方法
2. 掌握存储过程的调用方法
3. 掌握存储过程参数的使用方法
4. 掌握存储过程中变量和流程控制语句的使用方法

## 二、实验内容

### 题目一：创建无参存储过程

创建一个存储过程,查询 yggl 数据库 Employees 表中员工人数

### 题目二：创建带输入参数的存储过程

创建一个存储过程,查询 yggl 数据库 Employees 表中某个部门的员工信息,部门编号为输入参数

### 题目三：创建带输出参数的存储过程

创建一个存储过程,统计 yggl 数据库 Employees 表中某个部门的员工人数,部门编号为输入参数,员工人数为输出参数

### 题目四：创建带条件判断的存储过程

创建一个存储过程,根据输入的员工编号,判断该员工的工资等级：
- 收入 >= 3000：高薪
- 收入 >= 2000：中等
- 收入 < 2000：低薪

### 题目五：创建带循环的存储过程

创建一个存储过程,使用循环向一个测试表中插入指定数量的记录
