## 🗄️ 数据库说明

本课程主要使用以下数据库：

- **YGGL**：员工管理数据库,包含 `Employees`、`Departments`、`Salary` 三张表
- **test**：学生成绩管理数据库,用于进阶查询实验
- **netshop**：网上商城数据库,用于建表练习

## 📁 文件说明

每个实验文件夹中可能包含以下文件：

- `README.md` - 实验说明和答案
- `prepare.sql` - 实验准备脚本(创建表和插入数据)
- `Answer.sql` / `Ans.sql` - 实验答案SQL文件
- `temp.sql` - 临时测试文件

## 🛠️ 环境要求

- MySQL 5.7 或更高版本
- 推荐使用 MySQL Workbench 或命令行客户端
